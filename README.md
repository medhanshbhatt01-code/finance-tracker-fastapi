# Personal Finance Tracker API

A robust, asynchronous backend REST API for tracking personal expenses, managing budgets, and analyzing spending habits. Built using **FastAPI**, **PostgreSQL**, and **SQLAlchemy**.

## 🚀 Features
* **Expense Management:** Create, read, update, and delete expense logs.
* **Flexible Data Modeling:** Built with relational database principles using SQLAlchemy ORM.
* **Automatic API Documentation:** Interactive Swagger UI documentation generated out-of-the-box by FastAPI.

## 🛠️ Tech Stack
* **Framework:** FastAPI
* **Database:** PostgreSQL
* **ORM:** SQLAlchemy
* **Database Driver:** Psycopg2
